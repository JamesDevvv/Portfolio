WooCommerce to HubSpot Integration

WooCommerce → HubSpot (Webhook-based)
A webhook-driven integration that sends WooCommerce order and customer events to HubSpot in real time, enabling immediate contact upserts and deal creation in the CRM.

This integration listens for WooCommerce webhooks (order created/updated, customer created) and forwards normalized payloads to HubSpot via its REST API. Key considerations include webhook signature verification, idempotency (to avoid duplicate contacts/deals), and robust retry/error handling for transient failures.

My Role: Backend Developer – Implemented the webhook receiver and processing pipeline, created mapping logic from WooCommerce schema to HubSpot CRM objects, added signature verification and idempotency checks, and built monitoring & replay endpoints for failed events. Deployed the service in Docker and used PostgreSQL for logging and job tracking.

Key Features:
• Real-time order & customer sync via WooCommerce webhooks
• Signature verification and idempotency to prevent duplicates
• Automatic contact upsert and deal creation in HubSpot
• Retry mechanism and error logging with PostgreSQL for observability
• Admin endpoints for manual replay and audit of failed events
• Dockerized deployment for portability and easy scaling

Geolocation Application

React Frontend + Laravel Backend
A geolocation-based web application that captures user location, displays it on an interactive Leaflet map, and stores the data in a MySQL database.

This application leverages the browser’s Geolocation API to get the user’s current coordinates and renders them on a Leaflet-powered interactive map in React. Data is sent to a Laravel backend, where it is validated and stored in MySQL for history tracking and further analysis. Users can also view stored locations, filter by date, and manage their sessions securely.

My Role: Fullstack Developer – Implemented the React frontend with Leaflet map integration, created REST API endpoints in Laravel for location storage and retrieval, designed a MySQL schema for geolocation data, and added authentication & middleware protection for API routes.

Key Features:
• Real-time geolocation capture via browser API
• Leaflet.js interactive maps with markers and zoom
• Secure REST API for storing & retrieving location history
• MySQL database integration with Laravel Eloquent ORM
• Filters for viewing locations by date and user
• Authentication for protecting API endpoints
